# React Native Login Screen

## Clone

```Bash
$ git clone https://github.com/lucasfernandescwb/React-Native-Login.git
```
## Installing dependencies

```Bash
$ yarn
```
#### with npm

```Bash
$ npm install
```

## Running

```Bash
$ expo start
```
